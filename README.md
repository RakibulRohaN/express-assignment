# Express Assignment

Basic Express.js project to practice routing, GET/POST requests, query, headers, and JSON responses.

## Setup

```bash
npm install
npm start
```

Server runs on:

```
http://localhost:5000
```

## Routes

* `GET /` → Server Running Successfully
* `GET /welcome` → Welcome message
* `GET /student` → JSON data
* `GET /search?name=Monib&course=MERN` → Query data
* `POST /register` → JSON body
* `POST /product?id=101` → Query id
* `POST /login` → Headers (username, password)

## Tech

Node.js, Express.js
